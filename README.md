Quiniela World Cup Mexico 2026
Plataforma interactiva para el seguimiento en tiempo real de la quiniela familiar del Mundial 2026.

Descripción del Proyecto
Esta aplicación fue desarrollada para gestionar de manera automatizada y visual el marcador de puntos de la quiniela familiar. Permite visualizar de forma clara quién lidera la competencia y cómo se distribuyen los puntos entre los participantes.

Tecnologías Utilizadas
Python: Lenguaje principal para la lógica y procesamiento de datos.

Streamlit: Framework utilizado para la creación y despliegue de la interfaz web.

Pandas: Procesamiento y manipulación del archivo Excel de resultados.

Plotly: Generación de gráficas interactivas y dinámicas.

GitHub: Control de versiones y alojamiento del código.

Características
Visualización Dinámica: Gráfica de barras horizontales interactiva que destaca al líder actual.

Actualización Sencilla: Sistema basado en archivos Excel que facilita la actualización de marcadores.

Diseño Responsivo: Adaptable para consulta desde dispositivos móviles o computadora.
